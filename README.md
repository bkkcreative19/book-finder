# BookFinder App

## OverView

This app was created for pre-work for the Chingu Voyage 31 Program. It pulls data from the Google Books API.

**LIVE LINK** : https://mybookfinder.netlify.app/

## Features

-You type in any keyword or title of book and data is fetched based on that query.

-The data is then represented as cards with the book image, title, author, publisher, and a link to more information.

## Tech Used/Dependencies

-This app used Create-React-App

-Third Party Packages includes Sass and Axios

-Fonts are from Google Fonts
